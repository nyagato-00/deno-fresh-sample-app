# deno-fresh-sample-app
Freshを利用してWebアプリケーションを開発する際の練習用アプリケーションです。  
GitHub APIを利用して、任意のユーザーを検索することができます。

# Getting started

## denoのインストールとアップグレード

```
brew install deno
```

```
deno upgrade
```

# Usage
下記コマンドにて、アプリケーションを起動してください。

```
cd app
deno task start
```
